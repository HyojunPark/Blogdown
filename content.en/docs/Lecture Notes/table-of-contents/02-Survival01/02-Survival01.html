Nonparametric estimation

Data for nonparametric models

The following code creates a set of vector for survival analysis. It contains 5 individualsâ€™ survival time. \(1\) is an event (i.e., failure, death) and \(0\) is a cencored case.

tt <- c(7,6,6,5,2,4)
cens <- c(0,1,0,0,1,1)

Surv(tt,cens)
## [1] 7+ 6  6+ 5+ 2  4
aaa <- Surv(tt,cens) # demonstration only for checking how survival dataset was constructed

## [1] 7+ 6  6+ 5+ 2  4
